# 宁静海洋 - 网页启动页

一个清新极简风格的网页启动页界面，融合插画美学与交互设计，核心营造「宁静海洋」的视觉氛围。

## 功能特点

- **实时时间显示**：显示当前时间和日期
- **多搜索引擎支持**：支持Google、百度、必应和神马搜索
- **美观动画效果**：页面加载动画和交互动画
- **响应式设计**：适配不同屏幕尺寸
- **键盘快捷键**：Alt+1~4快速切换搜索引擎，Esc清空搜索框
- **自动聚焦**：页面加载时自动聚焦到搜索框
- **暗黑模式支持**：自动适应系统暗黑模式
- **移动端优化**：解决移动端100vh问题

## 技术栈

- HTML5
- CSS3 (动画、过渡效果、CSS变量)
- JavaScript (ES6+)

## 文件结构

## 使用方法

1. 克隆或下载本项目
2. 在浏览器中打开`index.html`文件
3. 可以将此页面设置为浏览器的启动页

## 快捷键

- **Alt + 1**: 切换到Google搜索
- **Alt + 2**: 切换到百度搜索
- **Alt + 3**: 切换到必应搜索
- **Alt + 4**: 切换到神马搜索
- **Esc**: 清空搜索框内容

## 自定义

- 更换背景：替换`background.png`文件
- 添加更多搜索引擎：在`main.js`中的`searchEngines`对象中添加
- 修改颜色：在`style.css`的`:root`变量中调整

## 设计理念

- **清新极简**：简洁的界面设计，减少视觉干扰
- **宁静海洋**：通过背景图片和配色营造宁静舒适的氛围
- **流畅交互**：添加适当的动画效果，提升用户体验
- **性能优化**：使用防抖函数优化性能，确保流畅体验
